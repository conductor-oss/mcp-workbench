
# Changelog

All notable changes to the **MCP Workbench** will be documented in this file.

## [1.0.0] - 2026-01-01

### Added
- **Initial Release**: Complete visual debugger for Model Context Protocol (MCP) servers.
- **Strict Protocol Support**: Exclusive focus on the "Streamable HTTP" transport for stateless, POST-only communication.
- **Protocol Inspector**: Real-time visualization of JSON-RPC traffic (`initialize`, capability discovery, etc.).
- **Tools Runner**: Dynamic form generation from JSON Schemas for manual tool testing.
- **Resource Explorer**: Browsing and inspection of server-side resources.
- **Prompt Tester**: Template-based testing for server-provided prompts.
- **Advanced Auth**: Support for OAuth 2.0 (Authorization Code Flow) and API Keys.
- **Multi-Server Management**: Save and persist multiple server configurations.
- **Branding**: Official Orkes identity integration (Logo, Slack, GitHub).
- **Tooling**: License header management and automated AWS Amplify deployment workflows.
- **AWS Hosted**: Deployment scripts for AWS Amplify and CI/CD via GitHub Actions.
